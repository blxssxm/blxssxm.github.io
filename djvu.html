The DjVU file format

What is DjVU and why should I use it>

DjVU is a file format that can be used for documents. Unlike the more popular .pdf file format, a .djvu file can often have a much smaller size and still retain its quality.

Opening a .djvu file

On Linux you can easily open them with zathura-djvu
